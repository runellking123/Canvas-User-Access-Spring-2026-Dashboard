CANVAS LMS USER ACCESS DASHBOARD - AI DATA PREPARATION

OVERVIEW:
This model tracks student engagement in Canvas LMS for Wiley University. Data includes login history and course activity logs. Always use measures from _Measures table, not raw columns.

DATA MODEL STRUCTURE:

TABLE 1: LastUserAccess
Purpose: Stores the most recent login timestamp for each Canvas user.
Grain: One row per unique user.
Primary key: user id
Use for: Login tracking, inactive student identification.

TABLE 2: UserCourseAccessLog
Purpose: Stores detailed course activity records showing what content users viewed and interacted with.
Grain: One row per user per course per content item.
Foreign key: user id (links to LastUserAccess)
Use for: Engagement analysis, content usage patterns, participation tracking.

TABLE 3: _Measures
Purpose: Contains all calculated measures for this model.
IMPORTANT: Always use measures from this table instead of raw column calculations.

HIDDEN TABLES (DO NOT USE):
- CanvasConfig: API configuration only.
- TriggerNewReports: API trigger only.

CRITICAL INSTRUCTION - ALWAYS USE MEASURES:

CORRECT: Use [Student Count] measure
INCORRECT: Count of user id column

CORRECT: Use [Total Views] measure
INCORRECT: Sum of times viewed column

CORRECT: Use [Active Students 30 Days] measure
INCORRECT: Count of students with recent login

BUSINESS DEFINITIONS:

Active Student: A student who has logged into Canvas within the specified timeframe (7 or 30 days).
Inactive Student: A student who has not logged into Canvas for over 30 days.
Never Accessed: A student who exists in Canvas but has never logged in.
View: Each time a student loads or displays content in Canvas counts as one view.
Participation: An active action such as submitting an assignment, posting to a discussion, or taking a quiz.
High Engagement: Student with 50 or more total content views.
Medium Engagement: Student with 10 to 49 total content views.
Low Engagement: Student with 1 to 9 total content views.
Zero Engagement: Student with 0 content views (at risk).

ENROLLMENT TYPE VALUES:
student - Enrolled learners taking courses for credit
teacher - Instructors and faculty who teach courses
ta - Teaching assistants who support instructors
observer - Parents, advisors, or auditors with view-only access
designer - Course designers who build content but do not teach

CONTENT TYPE VALUES:
assignments - Homework, projects, and submission activities
discussion_topics - Discussion boards and forum threads
quizzes - Tests, assessments, and quiz attempts
modules - Course content organizational containers
pages - Wiki pages and instructional content
attachments - Uploaded files, PDFs, and documents
announcements - Course-wide communications from instructors

COLUMN USAGE GUIDANCE:

FOR FILTERING ONLY (not calculations):
[enrollment type] - Filter to specific roles
[content type] - Filter to specific content types
[course name] - Filter to specific courses
[term name] - Filter to specific academic terms

FOR DISPLAY IN TABLES (not calculations):
[user name] - Display user names in detail tables
[course name] - Display course names
[last access at] - Display last login timestamp

NEVER USE FOR CALCULATIONS:
[times viewed] - Use [Total Views] measure instead
[times participated] - Use [Total Participations] measure instead
[user id] for counting - Use appropriate count measure instead

RESPONSE GUIDELINES:

For student counts: use card visuals with appropriate measure.
For comparisons: use bar or column charts with measures as values.
For distribution: use pie or donut charts with measures.
For percentages: use gauge visuals or card visuals.
For details: use table visuals with display columns and measures.
For role filtering: apply filter on enrollment type column.
For content filtering: apply filter on content type column.
Always use measures from _Measures table. Never calculate from raw columns.

FORMATTING STANDARDS:
Percentages: Display as percentage format with one decimal (75.3%)
Counts: Display as whole numbers with thousands separator (1,234)
Averages: Display with one decimal place (45.2)
